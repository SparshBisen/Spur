# Frontend - Spur AI Chat Widget

This is the React frontend for the AI Live Chat Support Agent.

## Features
- Clean chat UI with user/AI message distinction
- Auto-scroll to latest messages
- Typing indicator while AI is responding
- Session persistence using localStorage
- Input validation
- Error handling
- Responsive design

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

The app will run on http://localhost:3000

## Configuration

The backend API URL is configured in `src/App.js`. By default, it connects to `http://localhost:3001`.

## Usage

1. Make sure the backend server is running first
2. Open the frontend in your browser
3. Start chatting with the AI support agent!

The AI can answer questions about:
- Shipping policy
- Return/refund policy
- Support hours
- Payment methods
- Products
- And more!
